A lightweight Chrome extension that analyzes any website’s privacy practices and generates a simple, easy‑to‑understand Privacy Score. Built to help users quickly understand how safe and privacy‑friendly a website is.

✅ Features Detects whether the site uses HTTPS

Checks for a Privacy Policy link

Counts third‑party scripts

Detects common tracking domains

Computes a 0–100 privacy score

Clean popup UI with instant results

Runs automatically on every website you visit
